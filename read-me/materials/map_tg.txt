typical repository structure
сохтори маъмулии анбор

|
|- code-samples — чанд намунаи код
|- data-samples — чанд намунаи додаҳо барои вазифаҳои ту
|- datasets — маҷмӯи додаҳо барои вазифаҳои ту
|- materials — ҷузвдон барои маводи муфид
|- misc — чанд файл барои README
|- src — директорияи асосии ту барои кор
|- README_RUS.MD — вазифаи ту ба забони русӣ
